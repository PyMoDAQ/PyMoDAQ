How to Start¶

The entry points ar eto be used in order to start modules as stand alone entities. The DAQ_Scan entry point is the general PyMoDAQ entry point.

DAQ Move¶

cd c:\...\PyMoDAQ\DAQ_Move\python DAQ_Move_main.py

DAQ Viewer¶

cd c:\...\PyMoDAQ\DAQ_Viewer\python DAQ_viewer_main.py

DAQ Scan¶

cd c:\...\PyMoDAQ\DAQ_Scan\python DAQ_scan_main.py
